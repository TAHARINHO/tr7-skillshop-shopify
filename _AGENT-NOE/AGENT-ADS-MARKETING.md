# SECTION 2 — MODULES D'EXPERTISE

## MODULE 1 — STRUCTURE CAMPAGNES META ADS

### 1.1 Architecture Compte Meta (2026)

```
NIVEAU CAMPAGNE
├── 🔵 TOP OF FUNNEL — Acquisition froide
│   ├── Ad Set 1 : Audiences larges (Advantage+)
│   ├── Ad Set 2 : Intérêts ciblés [niche]
│   └── Ad Set 3 : Lookalike 1% (email liste / acheteurs)
│
├── 🟡 MIDDLE OF FUNNEL — Retargeting chaud
│   ├── Ad Set 1 : Visiteurs 30 jours (vue 3 produits+)
│   ├── Ad Set 2 : Abandons de panier 14 jours
│   └── Ad Set 3 : Vues de vidéo 50%+
│
└── 🔴 BOTTOM OF FUNNEL — Reconversion
    ├── Ad Set 1 : Acheteurs 90-180 jours (cross-sell)
    └── Ad Set 2 : Winback (inactifs 180 jours+)
```

### 1.2 Structure de Créatifs — 3:2:2 Rule

Pour chaque ad set, tester :
- **3 Hooks différents** (les premières 3 secondes)
- **2 Formats** (vidéo courte 15s + image statique)
- **2 CTA** ("Acheter maintenant" vs "Découvrir")

### 1.3 Hooks Gagnants par Type

**Hook PAIN** : "Tu en as marre de [douleur] ?"
**Hook RESULT** : "Voici comment [ICP] a [résultat] en [temps]"
**Hook QUESTION** : "Savais-tu que [fait surprenant sur la niche] ?"
**Hook PROOF** : "[Nombre] clients ont déjà [résultat]"
**Hook CURIOSITY** : "La raison pour laquelle [comportement contre-intuitif]..."
**Hook CONTROVERSY** : "Pourquoi [croyance commune] est fausse"
**Hook UGC** : "[Personne réelle] : 'Honnêtement je pensais pas que ça marcherait...'"

### 1.4 Template Créatif Video (15-30s)

```
[0-3s]  HOOK visuel + texte accrocheur (stopper le scroll)
[3-8s]  PROBLÈME — Identification avec la douleur ICP
[8-15s] SOLUTION — Le produit en action, bénéfice visible
[15-22s] PREUVES — Avis client, avant/après, stats
[22-28s] CTA — "Lien en bio" / "Obtenir le mien" / offre limitée
[28-30s] LOGO + Tagline
```

### 1.5 Message Match — Règle Absolue

```
AD HOOK = H1 LANDING PAGE = Premier bénéfice PDP

❌ MAUVAIS :
Ad : "Perd du poids rapidement avec notre legging"
Landing : "Collection Femme Printemps-Été 2026"

✅ BON :
Ad : "Ce legging m'a aidée à perdre 8kg en 3 mois ★★★★★"
Landing H1 : "Le legging qui sculpte et motive — plébiscité par 5000 femmes"
PDP Hook : "Ce que 5000 clientes ont en commun ? Ce legging."
```

## MODULE 2 — GOOGLE ADS E-COMMERCE

### 2.1 Structure Campagnes Google

```
PERFORMANCE MAX (priorité 2026) :
└── Asset Groups par thème produit
    ├── Asset Group 1 : Bestseller
    ├── Asset Group 2 : Collection [Niche 1]
    └── Asset Group 3 : Collection [Niche 2]

SHOPPING STANDARD (contrôle précis) :
├── Campagne 1 : Produits prioritaires (enchères élevées)
└── Campagne 2 : Catalogue long tail (enchères basses)

SEARCH (brand + concurrents) :
├── Campagne Brand : [Nom boutique] + variantes
└── Campagne Conquête : [Nom concurrent] (si budget)
```

### 2.2 Optimisation Flux Produits (Google Merchant Center)

Champs critiques pour le flux Shopping :
```
title : [Marque] [Couleur] [Produit] [Taille/Matière] [Usage]
        ex: "SweatFit Legging Gainant Noir Femme Sport Yoga Taille Haute"
description : [150-200 mots, mots-clés naturels]
google_product_category : [ID catégorie Google précis]
product_type : [Catégorie interne hiérarchique]
custom_label_0 : [Marge : High/Medium/Low]
custom_label_1 : [Bestseller : Top/New/Regular]
custom_label_2 : [Stock : InStock/LowStock]
```

## MODULE 3 — TIKTOK ADS

### 3.1 Formats TikTok E-Commerce

- **TopFeed In-Feed Ads** : 9-15s, native UGC style
- **Spark Ads** : Booster posts organiques performants
- **Shopping Ads** : Catalogue produits dans le feed
- **LIVE Shopping** : Vente en direct avec produits épinglés

### 3.2 TikTok Creative Direction

**Règle #1 : Native First**
Les meilleures TikTok Ads ressemblent à des TikToks normaux, pas à des publicités.

**Format gagnant TikTok E-Commerce :**
```
[0-2s]  SON + VISUEL choc (trending audio si possible)
[2-5s]  TEXTE QUESTION / HOOK en gros plan
[5-12s] DÉMONSTRATION produit (avant/après, unboxing, utilisation)
[12-18s] RÉACTION / PREUVE (avis, commentaires, UGC)
[18-22s] CTA + Offre (lien cliquable dans le texte)
```

### 3.3 Audiences TikTok

- **Intérêts** : [Niche principale] + [Complémentaires]
- **Comportements** : Acheteurs en ligne, Interacts with Shopping
- **Lookalike** : Sur base d'acheteurs existants (Pixel)
- **Broad** : Laisser l'algo optimiser (testé en 2026 = souvent meilleur)

## MODULE 4 — EMAIL & SMS MARKETING

### 4.1 Architecture Klaviyo Complète

```
FLOWS AUTOMATISÉS :
├── Welcome Series (5 emails)
├── Abandoned Checkout (3 emails + 1 SMS)
├── Browse Abandonment (2 emails)
├── Post-Purchase (5 emails)
├── Winback (3 emails)
└── Birthday (1 email)

CAMPAIGNS (envois manuels) :
├── Newsletter hebdo (produit phare + contenu)
├── Promotions (soldes, événements)
├── Lancements produits
└── Contenu éducatif (blog, tips)
```

### 4.2 Métriques Email (Benchmarks E-Commerce FR)

| Métrique | Mauvais | Bon | Excellent |
|---|---|---|---|
| Open Rate | < 15% | 25-35% | > 40% |
| Click Rate | < 1% | 2-4% | > 6% |
| Revenue/Email | < 0.05€ | 0.10-0.20€ | > 0.35€ |
| Unsubscribe Rate | > 0.5% | 0.1-0.3% | < 0.1% |

### 4.3 Stratégie SMS (Klaviyo / Yotpo SMS)

Usage SMS (taux d'ouverture 98%) :
- Abandon panier (dernier rappel)
- Flash sales (durée < 24h)
- Confirmation expédition
- Demande d'avis post-réception

Règle : Max 2-3 SMS/mois pour ne pas saturer.
